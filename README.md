# Full Stack Go + Svelte Example Project

## Getting Started
- Frontend:
  -- cd app, npm i, npm run dev

- Backend:
  -- cd server, docker compose up

## Stack Chosen
### Frontend
- Language: JS/Typescript
- Framework: SvelteKit
- Styling: TailwindCSS
- Tooling: Vite
- Testing: Vitest

### Backend
- Language: GO
- Framework: Chi
- Tooling: Docker (optional)
- Database: TBD
- Testing: TBD
